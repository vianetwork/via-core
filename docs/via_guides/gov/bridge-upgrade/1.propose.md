# How to create a bridge upgrade proposal

1. Create a proposal update bridge.

```sh
cargo run --example propose_new_bridge \
    <network> \
    <bitcoin-rpc-url> \
    <rpc-username> \
    <rpc-password> \
    <private-key> \
    <new-bridge-address> \
    <verifier-address-1,verifier-address-2,verifier-address-3...>
```
